# Hello_GIT

유닉스에 쬑끔 배우고 깃 활용 방법 배우고 깃 플로우, 깃헙 플로우에 대해 어떻게 하는지에 대해 배웠다.

여기에 기록하면서 README파일 작성하는것도 연습되고 좋은 방법이라고 생각한다.
